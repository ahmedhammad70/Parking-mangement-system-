# Parking Management System

This project is a Parking Management System developed using Java and follows the MVC (Model-View-Controller) architecture.

## Features
- Manage parking slots
- Add and remove vehicles
- Track vehicle entry and exit
- Organized using MVC design pattern

## Technologies Used
- Java
- Object-Oriented Programming (OOP)

## Concepts Used
- MVC Architecture
- OOP principles
- File handling

## How to Run
1. Open the project in IntelliJ IDEA or any Java IDE
2. Run `Main.java`

## Purpose
This project was built to improve my software development and system design skills as a Computer Science student specializing in Cybersecurity.
